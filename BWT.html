Instructions

- To input new codes insert bookwork code into the 'Add Bookwork code' section e.g. '3E'; letters should automatically capitalize

- Then insert the answer into the answer box and press add to send to storage; you can clear codes or copy them to clipboard here

- To view bookwork codes for checks you can either view storage or request the bookwork code such as '3E'

- Storage may be cleared if you clear cache!
